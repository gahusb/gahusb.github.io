# Emoji 이모티콘

* toc
{:toc .large-only}

 - 마크다운을 이용해 이모티콘을 표현 가능하다.
 - 깃허브도 문제없이 적용 가능하다.
 - 마크다운 문법에 :: 사이에 **이모티콘명**을 넣으면 자동으로 인식된다. :rocket:

```markdown

1. :tada:

:rocket: :metal:
```
